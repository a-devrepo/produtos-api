# API Node.js com Docker e Swagger

Esta é uma API desenvolvida em Node.js, estruturada para rodar em ambientes containerizados utilizando Docker e documentada com Swagger para facilitar o consumo dos endpoints.

## Tecnologias Utilizadas

* **Node.js**: Ambiente de execução Javascript assíncrono.
* **Docker**: Plataforma para criação e execução de containers.
* **Swagger (OpenAPI)**: Ferramenta para documentação e interface de testes da API.
* **Express**: Framework web para Node.js.

## Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:
* Node.js (v18 ou superior)
* Docker
